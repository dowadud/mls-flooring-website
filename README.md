## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/mls-flooring-website.git
cd mls-flooring-website
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is optimized for deployment on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will automatically detect Next.js and configure the build settings
4. Deploy!

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/mls-flooring-website)

## Customization

To customize the website:

1. **Colors:** Edit the Tailwind config or use Tailwind's color utilities in components
2. **Content:** Modify the text in `app/page.tsx`
3. **Images:** Replace images in the `public/` folder
4. **Styling:** Edit `app/globals.css` for global styles
